# Vision-Based Pan-Tilt Tracking System

## Overview

This project aims to design and build a vision-based pan-tilt tracking system capable of autonomously detecting and following moving objects in real time.

The project combines:

- Mechanical Design (SolidWorks)
- Computer Vision (OpenCV)
- Python Programming
- Mechatronics
- Servo Control
- Control Systems
- Rapid Prototyping

---

## Objectives

- Detect a moving hand using a webcam
- Determine the object's position in the camera frame
- Calculate pan and tilt commands
- Design and manufacture a two-axis pan-tilt mechanism
- Integrate servo motors with computer vision
- Evaluate tracking accuracy and response time
